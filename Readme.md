Website Bán Trà Sữa
Dự án website bán trà sữa được xây dựng bằng ASP.NET MVC, cung cấp giải pháp đặt hàng trực tuyến tiện lợi, quản lý sản phẩm chuyên nghiệp và theo dõi đơn hàng hiệu quả. Dự án đã hoàn thành 100% các chức năng cốt lõi.

🛠 Công nghệ sử dụng
Backend: ASP.NET MVC

Database: SQL Server (LocalDB)

ORM: Entity Framework 

🚀 Các tính năng đã hoàn thiện
Người dùng (Khách hàng)
Đăng ký/Đăng nhập: Quản lý tài khoản cá nhân an toàn.

Menu Trà Sữa: Xem danh sách sản phẩm, lọc theo loại (trà trái cây, trà sữa, kem cheese...).

Giỏ hàng: Thêm/xóa sản phẩm

Đặt hàng: Thực hiện quy trình đặt hàng trực tuyến ổn định.

Tương tác: Đánh giá sản phẩm và để lại phản hồi.

Wishlist: Lưu các món trà sữa yêu thích.

Quản trị viên (Admin)
Dashboard: Thống kê doanh thu và đơn hàng trực quan.

Quản lý sản phẩm: Thêm mới, chỉnh sửa, xóa các món trà sữa.

Quản lý đơn hàng: Theo dõi và cập nhật trạng thái đơn hàng (Đang chờ, Đang giao, Đã hoàn thành).

Tìm kiếm & Bộ lọc: Hỗ trợ tìm kiếm sản phẩm và lọc đơn hàng nhanh chóng.

📊 Cơ sở dữ liệu
Hệ thống sử dụng các bảng chính:

Users: Thông tin tài khoản người dùng và quản trị viên.

Products: Danh mục sản phẩm trà sữa.

Categories: Phân loại sản phẩm.

Orders & OrderDetails: Lưu trữ thông tin đơn hàng và chi tiết sản phẩm.

Comments & Wishlist: Dữ liệu tương tác từ người dùng.

⚙️ Hướng dẫn cài đặt
Clone repository: git clone <link-repo>

Cấu hình DB: Cập nhật ConnectionStrings trong web.config.
Import database mẫu
Chạy dự án: Nhấn F5 trong Visual Studio để khởi chạy website.
